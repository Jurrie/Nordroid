Nordroid

This is the website of Nordroid. Basically, it's more like a placeholder for the (mandatory) privacy policy.

Nordroid is meant to become a full Android port of Clavia's Nord Sound Manager.

Though it still has a long way to go. For now, this app only supports the Nord Electro 6D, because that's the only device I have access to.
